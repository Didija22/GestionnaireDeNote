# 📒 Mes Notes — Manuel d'utilisation

Petit guide pour utiliser l'application **Gestionnaire de Notes**.

---

## 1. Lancer l'application

1. Ouvrir le projet dans **Android Studio**
2. Attendre la fin de la synchronisation Gradle (barre en bas)
3. Brancher un téléphone (ou lancer un émulateur)
4. Cliquer sur **Run ▶️**

> L'application s'ouvre sur la liste des notes. Au premier lancement, elle est vide
> et affiche le message **« Aucune notes »**.

---

## 2. Créer une note

1. Appuyer sur le bouton **➕** en bas à droite
2. Une **palette de couleurs** apparaît → choisir une couleur
3. L'écran de création s'ouvre :
   - écrire le **titre**
   - écrire le **contenu**
   - (au besoin, changer la couleur avec les pastilles en haut)
4. Appuyer sur **Créer**

➡️ La note apparaît aussitôt dans la liste.

> ⚠️ Une note **vide** (sans titre) ne peut pas être enregistrée.

---

## 3. Les gestes sur une note

| Geste | Ce que ça fait |
|------|----------------|
| 👆 **Un appui** | Ouvre la note pour la **modifier** |
| 👆👆 **Deux appuis rapides** (double-clic) | Met la note en **favori** ⭐ ou l'enlève |
| 👆⏱️ **Appui long** | Ouvre le menu **Partager / Supprimer** |

---

## 4. Modifier une note

1. **Un appui** sur la note
2. Le formulaire s'ouvre **déjà rempli**
3. Changer le titre, le contenu ou la couleur
4. Appuyer sur **Modifier**

---

## 5. Les favoris ⭐

- **Double-clic** sur une note pour l'ajouter aux favoris (une étoile apparaît).
- Re-faire un **double-clic** pour la retirer.
- Le bouton **Favoris** (en haut à droite) n'affiche que les notes favorites.
  Appuyer une 2e fois dessus pour réafficher **toutes** les notes.

---

## 6. Rechercher une note 🔍

1. Écrire dans la barre **Rechercher** en haut
2. La liste se filtre automatiquement par **titre**
3. Effacer le texte pour revoir toutes les notes

---

## 7. Partager ou supprimer (bonus)

1. **Appui long** sur une note
2. Choisir :
   - **Partager** → envoie la note vers WhatsApp, mail, SMS, etc.
   - **Supprimer** → demande une confirmation, puis efface la note

---

## 8. Bon à savoir

- Toutes les notes sont **enregistrées dans le téléphone**. Elles restent là même
  après avoir fermé ou redémarré l'application.
- Les couleurs disponibles : 🟢 vert · 🔴 rouge · 🔵 bleu · 🟡 jaune · 🟠 orange · ⚪ gris.
